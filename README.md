## 🚀 Deploy

**Recommended:** connect this repo to [Vercel](https://vercel.com) or [Netlify](https://netlify.com) — it builds and deploys automatically on every push, no manual steps.

**Manual / static hosting:** run `npm run build`, then upload everything inside the generated `dist/` folder (not just `index.html` — it includes an `assets/` folder the page depends on).
